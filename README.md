## 🚀 About The Project
ImageGenius is a premium, Full-Stack Windows application designed specifically for Content Creators, Social Media Managers, and Web Developers. It eliminates hours of manual image processing by combining high-speed local Python image engines with cutting-edge Cloud AI.

From batch WEBP optimization to instant social media resizing and Groq-powered AI caption generation, ImageGenius is a complete creative microservice ecosystem.

**Note: The source code is housed in a private repository as this is a commercial SaaS product.**

## 🛠️ Built With
* **Python** (Core Logic & File Processing)
* **Pillow / PIL** (Advanced Image Manipulation & Resampling)
* **Groq LPU API** (Lightning-fast AI Caption Generation)
* **CustomTkinter** (Modern Dark-Mode Desktop GUI)
* **Flask & SQLAlchemy** (Cloud Auth & Database Backend)
* **PyInstaller** (Standalone .exe distribution)

## ✨ Key Features
* ⚡ **Auto-Optimize (WEBP):** Batch compresses heavy JPGs/PNGs into lightning-fast, web-ready WEBP files.
* 🎨 **Creator Mode (1-Click Socials):** Uses advanced padding algorithms (`ImageOps.pad`) to perfectly scale one image into YouTube, Instagram, Facebook, and LinkedIn formats instantly without cropping.
* ©️ **Watermark Engine:** Automatically stamps custom copyright text onto bulk photos with dynamic scaling.
* 🏷️ **SEO Batch Rename:** Instantly cleans and numbers hundreds of files for maximum Google Search optimization.
* 🤖 **AI Caption Generator:** Connects to Groq's LPU models to read user keywords and generate viral, hashtag-optimized social media captions in under 0.5 seconds.
* ☁️ **Cloud Auth & Webhooks:** Real-time license verification checking Free, Pro, and Elite tiers via integrated Gumroad/Xendit endpoints.
